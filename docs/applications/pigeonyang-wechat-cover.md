# Application: PigeonYang WeChat Article Covers

This is the first private application of Creator Cover Paradigm Distiller.

It uses the same three-layer system:

1. Router selects a distilled child skill for the article hook.
2. Production preserves the private PigeonYang anime identity and WeChat canvas.
3. `coverctl.py` or MCP enforces prompt, generation, output, and dimension gates.

This application is intentionally narrower than the product:

- platform: WeChat public account article cover first,
- default canvas: `2350x1000`, `2.35:1`,
- safe area: central square-safe zone `x=675..1675`,
- identity reference:
  `J:\PigeonYang\cover-style-distiller\cover-projects\_identity\pigeonyang-character-reference.png`.

Do not use this application workflow as the universal product definition.
